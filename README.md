# fivem-story-rp-core

Fivem Story RP Core - a modular resource providing core gameplay systems (survival, missions, cutscenes, utils) for GTA V FiveM servers.

This repository is a starter template with a structured file layout and placeholder modules for server and client logic.

## Features
- Modular client & server architecture
- Survival mechanics (hunger, thirst, radiation, sanity)
- Missions system
- Cutscene system with NUI creator
- Shared configuration and utilities

## Quick start
1. Place this resource into your server `resources` folder
2. Add `ensure fivem-story-rp-core` to your server.cfg
3. Configure stuff in the `config` folder
